# New Year Sales Analysis

## Project Overview

This project analyzes New Year sales data to identify customer purchasing behavior, sales trends, and business insights using Python.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset Features

- Customer Information
- Gender
- Age Group
- Marital Status
- Occupation
- State
- Product Category
- Orders
- Amount

## Key Findings

- Female customers contribute the highest revenue.
- Age group 26–35 generates maximum sales.
- Uttar Pradesh, Maharashtra, and Karnataka are top-performing states.
- IT Sector, Healthcare, and Aviation are top occupations.
- Food, Clothing & Apparel, Electronics & Gadgets, and Footwear & Shoes are leading product categories.

## Project Files

- New_Year_Sales_Analysis.ipynb

## Conclusion

The analysis helps businesses understand customer behavior and optimize marketing strategies to improve sales performance.
